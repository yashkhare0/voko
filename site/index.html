Why voko?

Interactive Setup

Easy-to-use wizard guides you through configuring languages, engines, and paths in seconds.

Multi-Engine Support

Choose your preferred translation provider: Google Translate, DeepL, LibreTranslate, or Yandex.

Smart Sync

Automatically translates missing keys. Use --strict mode to keep locale files clean and synchronized.

Framework Agnostic

Works with React, Vue, Svelte, Node.js, or any project using JSON locale files.

Get Started

Run directly with npx or install globally.

npm install -g voko
npx voko init
