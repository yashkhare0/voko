# Contributing to voko

Thank you for your interest in contributing to `voko`! We welcome contributions from the community.

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Install dependencies**:
    ```bash
    npm install
    ```

## Development

- **Build**: `npm run build`
- **Watch**: `npm run dev`
- **Test**: `npm test`

## Workflow

1.  Create a new branch for your feature or fix: `git checkout -b my-feature`.
2.  Make your changes.
3.  Ensure tests pass: `npm test`.
4.  Commit your changes with clear messages.
5.  Push to your fork and submit a Pull Request.

## Code Style

- We use TypeScript.
- Please ensure your code is clean and readable.
- Add tests for new features.
